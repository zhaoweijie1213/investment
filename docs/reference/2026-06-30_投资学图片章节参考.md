## 2. 章节摘要与操作参考

### 2.1 2019-08-24：左脚入，右肩出

**来源定位**：`Document/1.投资学/投资学上.pdf`，第64页起；核心操盘原则在第70页附近。

**核心意思**：

- “左脚入、右肩出”是一个反人性的技术纪律：不要抢在不确定底部，也不要幻想卖在最高点。
- “左脚”强调确认底部区域之后进入；“右肩”强调头部确认后离场，而不是在最兴奋时恋战。
- 这个原则适合和仓位分批配合：不追求鞋底买入，也不追求头顶卖出，接受腿部/腰部的现实操作。

**可操作化**：

| 情景 | 动作 |
| --- | --- |
| 刚下跌、还在右脚/右腿不确定区 | 忍，不急着重仓 |
| 有底部确认、量能配合、周期位置低 | 分批试仓 |
| 上涨进入右肩、利好密集、估值/情绪过热 | 卖交易仓或逐步降仓 |
| 买早或卖早 | 不自责，按下一轮周期和空间重新定位 |

### 2.2 2019-11-02 + 下册《投资的一般性原则》：十大原则与三三四

**来源定位**：

- `投资学上.pdf` 第166页起：2019-11-02《宏观经济形势、基本投资原则及区块链》。
- `投资学下.pdf` 第512页起：《投资的一般性原则》；第524页明确出现“十大原则，三三四的原则”。

**核心意思**：

- 投资原则分为两大块：选股逻辑和操作逻辑。
- 先做空间选择：地区、行业、行业龙头、时机；如果没有合适目标，就休息。
- 不替别人做作业，也不让别人替自己做作业。原则是讲方法，不直接给代码依赖。
- “三三四”在原文中被作为应坚持的原则提到，但 PDF 当前页没有展开完整定义；后续仓位执行先按用户确认口径使用：3成核心仓、3成交易仓、4成现金/备用仓。

**可操作化**：

| 维度 | 问题 |
| --- | --- |
| 地区 | 股票涨时，计价货币/汇率是否也有利？ |
| 行业 | 是否处在政策、周期、资本流向共同支持的位置？ |
| 龙头 | 是否有规模、流动性、治理和盈利支撑？ |
| 时机 | 现在是在左脚、左腿、腰部，还是已经右肩？ |
| 仓位 | 是否保留足够现金，不因单一叙事打满？ |

### 2.3 2019-11-30：一板斧/MACD

**来源定位**：`投资学上.pdf` 第214页起。

**核心意思**：

- 这章先讲主权货币估值逻辑，再转入技术工具。
- “三板斧”先只教“一板斧”：用 MACD 与成交量观察底部、趋势和主力进入迹象。
- 技术工具不是替代宏观和行业判断，而是在区域、产业、个股选择后提供入场节奏。

**可操作化**：

| 工具 | 用法 |
| --- | --- |
| 月线 MACD | 看大级别底部和趋势是否反转 |
| 周线 MACD | 看中期进入区域，不必抢最低点 |
| 成交量/成交额 | 验证是否有资金进入 |
| 日线 | 只处理短期节奏，不能替代大周期判断 |

### 2.4 2019-12-07：一板斧续，剑宗与气宗

**来源定位**：`投资学上.pdf` 第223页起。

**核心意思**：

- “剑宗”偏技术和战术，“气宗”偏战略和基本功。
- 资金规模越大，越不能只靠技术；千万级更要走气宗路线，百万级需要气宗和剑宗兼顾，小资金更要熟练一板斧。
- 宏观、价值、产业判断是战略层；MACD、量能、买卖点是战术层。

**可操作化**：

| 资金/阶段 | 侧重点 |
| --- | --- |
| 小资金 | 技术纪律更重要，避免一开始就错 |
| 中等资金 | 宏观方向 + 技术入场都要做 |
| 大资金 | 首先避免战略性错误，不能靠短线技术硬打 |

### 2.5 2021-01-02：投资中的“三观”

**来源定位**：`投资学下.pdf` 第315页起。

**核心意思**：

- 投资中的“三观”可整理为宏观、中观、微观。
- 宏观看资本在不同资产类别之间的流动；中观看产业和区域；微观看技术和个股。
- 没有宏观和中观，单靠微观技术容易变成盲打。

**可操作化**：

| 层级 | 对应问题 |
| --- | --- |
| 宏观 | 资金从债券、股票、地产、商品、现金流向哪里？ |
| 中观 | 哪个产业、哪个区域处于上升周期？ |
| 微观 | 具体标的是否有技术确认、量能确认、估值空间？ |

### 2.6 2021-01-16：投资中时间的意义 + 大宗

**来源定位**：`投资学下.pdf` 第336页起。

**核心意思**：

- 投资按时间可分战略级、战役级、战术级。
- 战略级以十年为单位；战役级以年为单位；战术级以天/月为单位。
- 大宗商品要先做价格分区，只有在极高或极低区间才有更高胜算。

**可操作化**：

| 时间尺度 | 典型动作 |
| --- | --- |
| 战略级 | 选长期资产方向，不频繁改主线 |
| 战役级 | 根据政策、周期、产业切换仓位 |
| 战术级 | 做交易仓、做T、降低成本 |

### 2.7 2021-01-30：投资中的空间意义

**来源定位**：`投资学下.pdf` 第355页起。

**核心意思**：

- 投资不仅是选股票，也是选空间：货币、地区、制度、产业、资产类别。
- 一个资产上涨，但计价货币或区域风险抵消收益，也不算好投资。
- 空间意义要求把个人、资产、市场放进更大的系统中看。

**可操作化**：

| 空间维度 | 检查点 |
| --- | --- |
| 货币空间 | 人民币、美元、港币等计价风险如何？ |
| 市场空间 | A股、港股、美股、商品市场哪个更有优势？ |
| 行业空间 | 资金是否从旧周期流向新周期？ |

### 2.8 2021-02-27：投资管理/投资清理

**来源定位**：`投资学下.pdf` 第396页起。

**核心意思**：

- 投资管理重点是“清理”：失败后的灾难现场处理、焦灼期整理、盈利但迷茫后的再出发。
- 行家需要定期清理：检讨结构、重建结构、处理错误、做中长期规划。
- 清理不是割肉情绪，而是让账户重新回到可管理状态。

**可操作化**：

| 清理对象 | 动作 |
| --- | --- |
| 失败仓位 | 判断是逻辑错、节奏错，还是仓位错 |
| 焦灼仓位 | 降低波动、拆分核心仓和交易仓 |
| 结构失衡 | 卖出高风险超配，恢复仓位纪律 |
| 中长期规划 | 保留现金和备用仓，不被短线拖死 |

### 2.9 2021-03-13：心态调整、战略选择和战术调整

**来源定位**：`投资学下.pdf` 第415页起。

**核心意思**：

- 投资不只考知识和智商，更考心态。
- 做出战略选择后，还要根据 A股市场周期、行业周期、汇率风险、通胀风险做战术调整。
- 不追求完美，有八九成把握就行动；避免“不悔、不怨、不等”的反面状态。

**可操作化**：

| 变量 | 调整方式 |
| --- | --- |
| 市场周期变了 | 降低交易仓或换节奏 |
| 行业周期变了 | 调整行业权重 |
| 汇率风险上升 | 控制跨境资产敞口 |
| 通胀风险上升 | 观察资源、黄金、能源链 |

### 2.10 2021-03-27：投资小结，避免投资失误的原则

**来源定位**：`投资学下.pdf` 第433页起。

**核心意思**：

- 投资是生活的一部分，不应该吞噬全部生活。
- 目标不是人人成为股神，而是掌握生产力工具，获得财富自由的可能。
- 重要句式：圣人明势，君子知止，大师度危，高手从权。

**可操作化**：

| 原则 | 含义 |
| --- | --- |
| 明势 | 先判断大趋势和大环境 |
| 知止 | 知道何时停止加仓、何时退出 |
| 度危 | 能识别并穿越风险区 |
| 从权 | 不固执，根据形势灵活调整 |

### 2.12 2023-03-11：投资学补讲1，态势感知

**来源定位**：`投资学下.pdf` 第465页起。

**核心意思**：

- 四堂补课前两堂讲“知”：第一堂是态势感知，第二堂是蛛丝马迹。
- 态势感知要把周期、空间、资本流动放在一起看。
- 不能只盯价格，要看到利率、税收、资产价格、资本流向这些更上游的变量。

**可操作化**：

| 感知对象 | 问题 |
| --- | --- |
| 周期 | 当前处在基钦、朱格拉、康波哪个节奏？ |
| 空间 | 资金从哪里来、往哪里去？ |
| 资本流 | 利率、税收、资产价格是否改变资金方向？ |

### 2.13 2023-03-25：投资学补讲2，蛛丝马迹

**来源定位**：`投资学下.pdf` 第474页起。

**核心意思**：

- 蛛丝马迹是对态势感知的细化：从细微信号中判断资本方向和产业轮动。
- 关键变量包括资本积累率、实质负利率、区域性资本流动、产业轮动。
- 这章强调“知”的能力：先看懂，不急着做。

**可操作化**：

| 信号 | 解释 |
| --- | --- |
| 资本积累率高 | 更容易吸引资本流入 |
| 实质负利率 | 推动资产从现金/债券向其他资产流动 |
| 产业轮动 | 一个周期的弃子可能成为下个周期的网红 |

### 2.14 2023-04-08：投资学补讲3，起于暗淡

**来源定位**：`投资学下.pdf` 第484页起。

**核心意思**：

- “起于暗淡”讲入场：真正好的机会往往在不热闹、被忽视、被抛弃时出现。
- 投资是一场战役，必须想清楚方式、规模、时间。
- 暗淡包括黑马、周期弃子、政策宠儿等类型。

**可操作化**：

| 暗淡类型 | 含义 |
| --- | --- |
| 黑马 | 早期被低估，未来有巨大成长空间 |
| 周期弃子 | 被上一个周期抛弃，可能成为下一周期焦点 |
| 政策宠儿 | 本身未必完美，但政策风口出现 |

### 2.15 2023-04-22：投资学补讲4，止于辉煌

**来源定位**：`投资学下.pdf` 第494页起。

**核心意思**：

- “止于辉煌”讲退出：开始就是为了辉煌的结束，结束就是为了光明的开始。
- 不能只看股价线，要把公司/资产看成系统中的能量体：膨胀、边际、耗散、转换。
- 真正的退出常发生在系统能量最亮、市场最兴奋、叙事最完整的时候。

**可操作化**：

| 阶段 | 动作 |
| --- | --- |
| 刚起于暗淡 | 建仓/观察 |
| 中段扩张 | 持有核心仓，做交易仓 |
| 辉煌阶段 | 逐步退出交易仓，防止恋战 |
| 系统转向 | 换下一轮暗淡 |

### 2.16 2024-04-20：投资三忌

**来源定位**：`投资学下.pdf` 第503页起。

**核心意思**：

- 这堂课从“不能怎样做”入手，强调投资中的三种错误：搭错车、不知止、不专精。
- 它与“态势感知、蛛丝马迹、起于暗淡、止于辉煌”构成后续补课框架。
- 关键是摆脱主观臆断，用宏观规律和仓位纪律来约束自己。

**可操作化**：

| 三忌 | 解释 |
| --- | --- |
| 搭错车 | 选错周期、行业、资产、货币空间 |
| 不知止 | 不知道何时停止加仓或何时退出 |
| 不专精 | 没有自己的系统，今天听这个、明天追那个 |

## 3. 和当前油股操作的连接

| 投资学原则 | 当前油股对应 |
| --- | --- |

| 三观 | 宏观看油价/美元/库存，中观看油气链，微观看中海/中曼价位。 |
| 投资清理 | 当前浮亏仓位要拆分核心仓和交易仓，而不是情绪清仓或机械加仓。 |
| 起于暗淡 | 油气链若仍在长期重估逻辑中，低迷区是观察和分批区。 |
| 止于辉煌 | 若油价和油股进入热闹叙事，交易仓要主动卖出降成本。 |
| 投资三忌 | 防止搭错节奏、不知止、只盯单一股票不看系统。 |

# 金油同跌、科技股与美元走强下的资本三流分析

> 时效提示：本文写于 2026-06-26。行情、利率、油价、金价、美元指数、科技股价格均需用最新报价复核。本文是宏观研究框架，不构成买卖建议。

## 1. 分析结论

1. 2026-06-26 这轮“黄金、白银、原油同时回落，科技股相对强，美元偏强”的组合，不应解释为“全球需求突然同步塌陷”，更像是“避险溢价、战争溢价和通胀溢价同时被压缩，资金暂时回到美元现金、短债、美债和 AI/科技龙头叙事”。
2. 原油下跌的核心是霍尔木兹重开后的断供风险溢价回吐；贵金属下跌的核心是实际利率上行、美元流动性偏紧和高位拥挤仓位兑现。两者同跌，不要求需求同步减弱，只要求“风险溢价被快速拿掉 + 资金需要流动性”。
3. 科技股和 AI 赛道在此时走强，说明资金并没有完全离开风险资产，而是在风险资产内部从资源、贵金属、油气弹性资产切向“美元计价、流动性最好、融资叙事最强、指数权重最大”的资产。
4. 资本三流拆解为：流量层面，美元融资和短端安全资产吸收资金；流向层面，资金从金油和资源股流向美元、美债、科技龙头和现金；流速层面，霍尔木兹重开、停火/协议消息和实际利率变化导致风险溢价快速重定价。
5. 这与仓库既有框架并不矛盾：卢麒元框架中的“金转油/油粮”是中长期价值中枢迁移，但“向心坍缩/速冻”阶段会出现所有高波动资产先被卖出、美元现金和美国核心资产暂时变强的反身性过程。

## 5. 资本三流模型

### 5.1 定义

| 维度 | 定义 | 观察指标 | 当前含义 |
| --- | --- | --- | --- |
| 流量 | 市场中可用于买资产的美元资金总量和融资条件 | Fed 资产负债表、TGA、逆回购、货币基金、短端利率、信用利差 | 流量没有明显宽松，资金更偏向安全美元资产 |
| 流向 | 资金买入和卖出的资产类别 | ETF 流入流出、板块涨跌、股债汇商相对强弱 | 从金油/资源/外围资产流向美元、美债、AI/科技龙头 |
| 流速 | 资金切换速度和风险溢价压缩速度 | VIX、MOVE、期货成交、期限结构、日内跳空、基差 | 霍尔木兹重开和利率变化让金油溢价快速回吐 |

### 5.2 当前“钱去哪了”

| 资金去向 | 证据链 | 解释 |
| --- | --- | --- |
| 美元现金和短债 | 美元偏强、短端无风险收益仍高 | 资金先拿确定收益和流动性，不急着买高波动商品 |
| 美债和美元融资体系 | 10Y 名义利率仍在 4% 以上，财政融资仍需吸收资金 | 全球美元资产仍是危机时的主承接池 |
| AI/科技龙头 | 科技股相对强，AI 仍是美元资产中最有融资叙事的赛道 | 资金不是完全避险，而是从资源切到高流动性成长资产 |
| 美国大盘指数 | 指数权重集中于科技和平台公司 | 被动资金和主动资金都更容易买指数，而不是买小盘资源 |
| 没有明显流向原油和贵金属 | 金油同跌、油气股承压 | 说明避险/断供/通胀仓位在去拥挤 |

## 6. 计算公式、计算过程与计算逻辑

下面公式不是“预测神器”，而是把直觉拆成可复核指标。每个指标都应滚动更新。

### 6.1 实际利率压力指数

**公式：**

```text
实际利率压力 = Δ10Y TIPS 实际利率 - Δ10Y 通胀预期
```

**含义：**

- 10Y TIPS 实际利率上升，黄金机会成本上升。
- 10Y breakeven 通胀预期下降，抗通胀资产溢价下降。
- 两者同时发生时，黄金最容易被压。

**以 FRED 6/16 至 6/24 附近数据估算：**

```text
6/16: 10Y TIPS ≈ 2.14%，10Y breakeven ≈ 2.29%
6/24: 10Y TIPS ≈ 2.23%，10Y breakeven ≈ 2.18%

Δ10Y TIPS = +0.09%
Δ10Y breakeven = -0.11%
实际利率压力 = +0.09% - (-0.11%) = +0.20%
```

**解释：**

0.20 个百分点的压力不小。它说明黄金面对的是“实际利率上行 + 通胀预期回落”的双重压制，所以即便美元长期信用有问题，短线黄金也会跌。

### 6.2 原油战争溢价回吐指数

**公式：**

```text
战争溢价回吐 = 高点Brent价格 - 当前Brent价格
战争溢价回吐率 = (高点Brent价格 - 当前Brent价格) / 高点Brent价格
```

**示例计算：**

若霍尔木兹紧张时 Brent 一度在 80-84 美元，重开后回到 72-75 美元：

```text
按 84 -> 73 估算：
战争溢价回吐 = 84 - 73 = 11 美元/桶
战争溢价回吐率 = 11 / 84 = 13.1%
```

**解释：**

这 13% 左右不一定来自真实需求走弱，而可能只是“断供保险费”消失。要确认是否变成需求衰退，需要看库存、裂解价差、炼厂开工、期限结构。

### 6.3 金油同步压力指数

**公式：**

```text
金油同步压力 = 实际利率压力 + 原油风险溢价回吐 + 美元流动性压力
```

为便于跟踪，可标准化为方向打分：

| 指标 | 方向 | 分数 |
| --- | --- | --- |
| 10Y TIPS 上行 | 压黄金 | +1 |
| 10Y breakeven 下行 | 压金油通胀仓位 | +1 |
| Brent 战争溢价回吐 | 压原油 | +1 |
| 美元指数/贸易加权美元上行 | 压美元计价商品 | +1 |
| 科技股相对强于资源股 | 资金离开商品仓位 | +1 |

```text
金油同步压力分数 = 上述有效信号相加
0-1：压力弱
2-3：中等压力
4-5：强压力
```

**当前判断：**

2026-06-26 附近至少满足实际利率上行、通胀预期回落、战争溢价回吐、美元偏强、科技股相对强等多项，所以金油同跌不是异常，而是组合压力下的结果。

### 6.4 AI 资金吸纳指数

**公式：**

```text
AI资金吸纳指数 = 科技股相对强弱 + 成交/资金流 + 利率敏感度修正 + 信用利差修正
```

简化跟踪：

```text
科技股相对强弱 = QQQ/SPY 的阶段收益率 - 能源股ETF/SPY 的阶段收益率
AI龙头相对强弱 = AI龙头篮子收益率 - SPY收益率
```

**解释：**

如果 QQQ、纳指、半导体和 AI 龙头强于能源、材料、贵金属矿业股，说明资金没有完全进入现金，而是进入了“美元体系内部的高流动性成长资产”。这就是“钱去了 AI/科技龙头”的价格证据。

### 6.5 资本三流综合打分

```text
资本三流总分 = 流量分 + 流向分 + 流速分
```

| 分项 | 计算逻辑 | 当前判读 |
| --- | --- | --- |
| 流量分 | 看美元流动性是否宽松：短端利率、TGA、逆回购、信用利差 | 未见明显宽松，偏紧 |
| 流向分 | 看资金买什么：美元、美债、AI、金油、资源股、外围资产 | 流向美元/美债/AI，流出金油和资源 |
| 流速分 | 看变化速度：VIX、MOVE、期货跳空、期限结构、美元急涨急跌 | 溢价压缩速度快，流速偏高 |

```text
若 流量偏紧 + 流向核心美元资产 + 流速加快：
结论 = 向心坍缩早期/局部速冻，而不是商品全面牛市加速。
```

## 10. 风险与反例

1. 如果金油同跌伴随 VIX、信用利差、美元融资利差快速上行，则这不是普通回调，而可能是流动性冲击，所有高波动资产都会被卖。
2. 如果 Brent 跌破 68、65，同时 EIA 库存连续累积、裂解价差走弱、炼厂开工下降，则“只是战争溢价回吐”的判断需要下修，需求衰退定价可能成为主线。
3. 如果 10Y TIPS 回落但黄金仍继续跌，说明黄金不是被实际利率压，而是被强制去杠杆或央行/ETF 流出压制。
4. 如果科技股开始补跌，而美元和美债继续强，说明资金从“AI/科技龙头”进一步撤到“纯现金/短债”，向心坍缩进入更高强度。
5. 如果美元突然走弱、实际利率回落、金油同步反弹，则当前这轮下跌可能只是短期仓位清洗。
6. 如果霍尔木兹重开只是名义重开，实际船流、保险、港口排队仍不正常，原油可能在短暂下跌后重新计入供应风险。

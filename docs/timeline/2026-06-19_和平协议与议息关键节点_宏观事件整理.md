# 美伊和平协议与美联储议息关键节点时间线

> 时效提示：写于 2026-06-16，6/16–17 FOMC 与 6/19 签约结果尚未发生，以下为节点与分支推演，须用最新公告复核。

## 1. 分析结论

1. 6/14–7/29 是一条连续验证链：协议预期（已抢跑）→ FOMC 措辞 → 6/19 签约能否落地 → 7/7 EIA 基本面 → 7/29 FOMC 是否兑现宽松。
2. 资本流速由 6/19 签约结果直接决定，分"签约/不签约"两条分支。
3. 油价已先行计入协议利好，但物流真实恢复滞后，价格与基本面可能短期背离。

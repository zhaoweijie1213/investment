# 原油转身与美元重置指标清单

## 1. 分析结论

1. 原油是否在7月转身，必须用“价格、库存、物流、政策、融资”五类指标共同验证。
2. 美元重置是否进入新阶段，重点看美国财政融资压力是否从油价冲击中独立出来。
3. 最重要的反向信号是：油价短跌但库存继续恶化，或油价回落但美债融资压力继续上升。

## 6. 可验证指标

| 观点 | 验证指标 | 数据变化说明什么 | 反向信号 |
| ---- | ---- | ---- | ---- |
| 原油真转身 | Brent近月价格连续回落，同时1-6月价差收窄 | 风险溢价和现货短缺同步缓和 | 价格回落但价差仍强 |
| 库存确认转身 | EIA原油和成品油库存降幅收窄或转增 | 供需压力缓和 | 库存继续大幅下降 |
| 物流确认转身 | 霍尔木兹船流恢复、保险费下降、等待油轮减少 | 供应可实际回归市场 | 船流低、保险费高、等待不减 |
| 需求端压制油价 | 中国/亚洲炼厂开工、美国汽油需求、航空燃油需求 | 油价下跌可能来自需求坏而非供给好 | 需求回升但油价仍跌，说明供给恢复更强 |
| 美元重置融资端 | 美国季度再融资、长债拍卖尾部、10Y/30Y收益率 | 融资压力是否上升 | 拍卖强、长端利率回落 |
| Fed确认路径 | FOMC声明和发布会中的能源、通胀、金融条件措辞 | 高油价是否推迟宽松 | Fed淡化能源冲击 |
| 黄金与原油切换 | 金油比、黄金ETF、Brent期限结构 | 资金是否从信用避险转向实物短缺 | 黄金和原油同涨，说明风险未解除 |

## 7. 风险与反例

1. 数据口径差异：EIA、IEA、OPEC、商业库存和船运数据可能给出不同方向，不能用单一口径确认转身。
2. 价格提前反映：油价可能先交易预期，库存和物流滞后确认。
3. 政策干预：战略储备释放、补贴、护航、制裁豁免可能短期改变价格。
4. 战争升级：任何冲突升级都可能覆盖供需数据本身。
5. 美元融资压力独立恶化：即使油价回落，美债供给压力也可能继续推动美元重置。

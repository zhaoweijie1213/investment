# Investment Macro Research Notes

本仓库用于整理宏观投资、美元体系、能源、贵金属、日元套息、AI/Token 美元、卢麒元周末聊天与闭门会相关研究。

## 当前核心框架

卢麒元闭门会和近期周末聊天的主线，不是单纯看多某个资产，而是围绕“资本载体”和“价值中枢”的迁移：

```text
美元旧锚松动（中国廉价商品 + 中东石油）
-> 黄金先表达美元贬值
-> 金银铜油粮次第表达
-> 日本/日元触发全球货币震荡
-> 霍尔木兹/中东作为通道之战反复控制油价
-> 资本向美国/日本等核心区回流并速冻
-> 向心坍缩
-> 美元重置与人民币电币复合
```

## 关键结论

1. “金转油”不能狭义理解为只买原油或单一股票。闭门会中“油”泛指能源链，包括油、气、煤、电、化工、绿电等。
2. 石油短线可能被协议消息、霍尔木兹开放叙事、日元套息拆仓或流动性冲击压住，但这不等于油粮/能源链的长期逻辑失效。
3. 7月不是单日拐点，而是验证窗口：看 EIA 库存、霍尔木兹船流/保险费、Brent 期限结构、FOMC 口径和日元套息平仓。
4. 日元套息交易结束是近期关键的资本三流信号：先看 USD/JPY、JGB、港股、韩股、台股、美股期指和 VIX。
5. 9月30日前，美国有强动机反复压油价和通胀预期，为美债融资、AI资产融资/上市窗口和风险资产估值争取时间；但能否压住取决于库存、航运、地缘和流动性。
6. 80 美元附近更像旧锚；若7月油价上冲后再被压，100-120 可能成为新的控制区；150 以上更接近危机定价区。
7. 向心坍缩不是单向上涨逻辑。速冻阶段很多资产和商品可能先跌，现金阶段性重要；之后才看能源、粮食、战略资源和人民币自定价。

## 9月30日前后的油价推演

这部分记录 2026-06-16 对话中从“所以在9月30日之前，油价又会被压制回去是吗？”之后形成的完整逻辑。

### 一句话结论

9月30日前更像美国的“压油价窗口”，但不是油价必然回到旧锚；如果7月油价上冲确认，8-9月即使再次压价，也更可能是在较高区间内压制通胀预期，而不是把油价重新压回80美元附近。

### 逻辑链

```text
美国财政与美债压力集中到9月30日前后
-> 需要维持发债、风险资产估值和AI融资/上市窗口
-> 高油价会推高通胀预期、压低降息概率、冲击科技股估值
-> 美国有动机通过协议消息、霍尔木兹开放叙事、库存口径、需求叙事和流动性工具反复压油价
-> 但若7月出现航运/保险/库存/期限结构共振，说明原油旧锚已松动
-> 8-9月压价仍可能发生，但压回80的难度上升，100-120可能成为新的控制区，150以上属于危机定价区
```

### 分阶段判断

| 时间区间 | 推理判断 | 关键前提 | 需要验证的数据 | 反向信号 |
| --- | --- | --- | --- | --- |
| 2026-07 | 油价可能进入压不住的验证窗口 | 霍尔木兹风险、船运保险、库存下降、期限结构转强同时出现 | Brent/WTI、EIA库存、VLCC运费、战争风险保险、Brent近远月价差 | 协议落地、航运恢复、库存累积、期限结构转弱 |

### 油价区间含义

| 油价区间 | 框架含义 | 观察重点 |
| --- | --- | --- |
| 80美元附近 | 更像旧锚和旧均衡区 | 如果7月上冲后还能稳定回到80附近，说明压制仍有效，金转油节奏需要放慢验证 |
| 100-120美元 | 可能成为新的控制区 | 说明油价旧锚松动，但美国仍可通过消息和流动性管理压制危机定价 |
| 150美元以上 | 更接近危机定价区 | 通常需要霍尔木兹、保险、库存、地缘升级或金融流动性风险同时共振 |

### 与AI/Token美元叙事的关系

8-9月若出现压油价、稳美债、稳纳指、强化AI融资/上市叙事的组合，可以理解为美国在为科技金融资产和Token美元叙事争取窗口。这个判断目前属于推理判断，不是已确认事实；需要用 OpenAI/Anthropic 等公司的官方融资、上市安排，以及美债利率、纳指、AI龙头股、美元流动性数据交叉验证。

### 证伪条件

1. 7月油价没有上冲，且EIA库存、期限结构、航运保险均不支持供给风险，说明“7月压不住”判断需要下修。
2. 8-9月油价顺利回到80附近且美债利率、通胀预期同步回落，说明美国压价能力仍强，100-120新控制区判断需要推迟。
3. 如果全球需求快速坍缩或美元流动性危机先爆发，油价可能先跌而不是直接进入上升浪。
4. 如果中东协议真实落地、霍尔木兹航运完全恢复、保险费回落，则油价危机溢价会被削弱。
5. 如果AI融资/上市没有明确时间表，不能把“压油价配合AI上市”当成事实，只能作为观察假设。

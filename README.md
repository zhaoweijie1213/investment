# Investment Macro Research Notes

本仓库用于整理宏观投资、美元体系、能源、贵金属、日元套息、AI/Token 美元、稳定币、卢麒元周末聊天、微博与闭门会相关研究。

本文是卢麒元框架的总索引。后续每次新增或更新卢麒元相关资料（微博、周末聊天、闭门会、课程摘录、访谈等），都要同步回到本文，重新整理一次完整逻辑：新增内容是否改变主线、资产排序、时间节点、验证指标和风险反例。

## 1. 一句话总框架

卢麒元的核心不是单纯看多黄金、石油或某只股票，而是用“资本载体与价值中枢迁移”解释美元旧秩序松动、全球资本回流核心区、资产价格重定价和新货币锚形成的过程。

```text
美元旧锚松动（中国廉价商品 + 中东石油）
-> 黄金先表达美元信用折价
-> 金银铜油粮次第表达资源与必需品重定价
-> 日本/日元触发全球套息交易平仓与货币震荡
-> 霍尔木兹/中东成为能源通道和美元石油锚的反复博弈点
-> 资本向美国/日本等核心区回流并出现速冻
-> 向心坍缩
-> 美元重置与人民币电币复合
```

## 2. 底层理论

| 层级 | 原文框架 | 经济含义 | 需要验证的事实 |
| --- | --- | --- | --- |
| 价值中枢 | 资本载体与价值中枢迁移 | 全球定价锚从旧美元体系向资源、能源、工业能力、数字金融和新信用载体重估 | 美元指数、美债需求、黄金、原油、战略资源、人民币汇率 |
| 美元旧锚 | 中国廉价商品 + 中东石油 | 低通胀商品供给和石油美元共同支撑美元购买力 | 中国出口价格、油价、OPEC/中东通道、美国通胀预期 |
| 资本三流 | 流量、流向、流速 | 看资金总量、去向和移动速度，而不是只看单日价格 | 美债拍卖、美元流动性、外资流向、港股/日股/美股成交、VIX |
| 向心坍缩 | 资本回流核心区并速冻 | 危机中资本先回流美元/日元等核心市场，外围和离岸资产承压 | USD/JPY、JGB、港股、韩股、台股、美债、离岸美元 |
| 美元重置 | 负债与资产重新匹配 | 美国试图通过利率、通胀、战争、数字资产和资源定价重塑美元信用 | Fed 路径、财政赤字、TGA、发债节奏、稳定币监管 |
| 人民币电币复合 | 绿电/工业用电与人民币信用 | 人民币可能从外汇约束转向工业能力、能源和电力信用约束 | 人民币汇率、绿电消纳、工业电价、跨境结算、政策立法 |

| 阶段 | 关键词 | 资产表达 | 作者倾向 | 最容易误读的地方 |

## 6. 9月30日前后的油价推演

9月30日前更像美国的“压油价窗口”，但不是油价必然回到旧锚。如果7月油价上冲得到航运、保险、库存和期限结构验证，8-9月即使再次压价，也更可能是在较高区间内压制通胀预期，而不是把油价稳定压回 80 美元附近。

```text
美国财政与美债压力集中到9月30日前后
-> 需要维持发债、风险资产估值和AI融资/上市窗口
-> 高油价会推高通胀预期、压低降息概率、冲击科技股估值
-> 美国有动机通过协议消息、霍尔木兹开放叙事、库存口径、需求叙事和流动性工具反复压油价
-> 但若7月出现航运/保险/库存/期限结构共振，说明原油旧锚已松动
-> 8-9月压价仍可能发生，但压回80的难度上升，100-120可能成为新的控制区，150以上属于危机定价区
```

| 油价区间 | 框架含义 | 观察重点 |
| --- | --- | --- |
| 80美元附近 | 更像旧锚和旧均衡区 | 如果7月上冲后还能稳定回到80附近，说明压制仍有效，金转油节奏需要放慢验证 |
| 100-120美元 | 可能成为新的控制区 | 说明油价旧锚松动，但美国仍可通过消息和流动性管理压制危机定价 |
| 150美元以上 | 更接近危机定价区 | 通常需要霍尔木兹、保险、库存、地缘升级或金融流动性风险同时共振 |

## 8. 风险和反例

1. 如果全球需求快速坍缩，油、铜、粮食等商品可能先跌，不能把资源重定价理解为单向上涨。
2. 如果中东协议真实落地、霍尔木兹航运恢复、保险费回落，原油危机溢价会被削弱。
3. 如果美国财政融资顺利、美债长端需求恢复、通胀预期下行，美元重置压力可能延后。
4. 如果日本央行继续压制利率、日元套息交易恢复，资本三流的流向/流速判断需要下修。
5. 如果中国政策、国储、人民币电币复合没有落地，不能把理论路径当成可交易事实。
6. 如果 AI/Token 美元叙事没有官方融资、监管和储备结构支持，不能把它当成已确认美元重置工具。
